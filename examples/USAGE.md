# Usage Examples

```bash
# List themes
npx ailogo themes

# Preview themes
npx ailogo themes --preview

# Render with config
npx ailogo init
npx ailogo show

# Override via flags
npx ailogo show --text "Hello" --theme claude --boxed --align center

# Custom gradient
npx ailogo show --text "AI" --color "#30cfd0" "#330867"
```
